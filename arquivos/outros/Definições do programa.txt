Campo origem das tabelas cartão, dinheiro e cheque.

0 - dívida avulsa;
1 - pagamento de compra;
2 - entrada avulsa;
3 - venda de produto.


Campo origem tabela despesas, ganhos.

0 - cartão;
1 - cheque;
2 - dinheiro,parcelado.
3 - dinheiro,avista.

campo id_origem se refere ao id da operação.

campo status tabela despesas.

0 - aberta;
1 - paga;
2 - atrasada.

campo origem tabela his_entradas.

0 - entrada avulsa;
1 - compra;

Campo origem tabela jus_cancelamento_nota.

0 - nota de venda;
1 - nota de compra;

campo status nas tabelas compra e venda.
0 - aberta;
1 - cancelada;
